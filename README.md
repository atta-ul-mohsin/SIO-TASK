
# SIO Time Logger


- make sure following softwares are installed on the system
    - docker
    - npm
    - composer





- Run following commands in sequence

    - git clone https://github.com/atta-ul-mohsin/php-laravel-nginx-mysql.git

    - cd into cloned dir

    - composer install

    - docker-compose up -d

    - npm install
    - npm run dev


- access the system on this url
  http://localhost:8000/
